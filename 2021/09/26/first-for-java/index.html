Java初学第二天

算术运算符、赋值运算符、比较运算法

PS：对于赋值符号=，程序从右向左，对于算术运算符程序从左向右进行；

大体同c一样，但有几点需要注意：

① += （*=等同理）以及 ++ （–） 不改变数据类型
②（a++）与 （++a）在c++和Java中不一样

c++中运算会先执行（）里面的命令，但是（a++）则是在执行完这一整条语句后再进行a += 1；

例 ： b = a + (a++) + (++a);//6

先进行（a++）但是a += 1等执行完b = a + (a++) + (++a);再进行；

然后进行（++a），执行完后，此时a = 2；

然后在执行b = a + a + a = 2 + 2 + 2 = 6；

Java中运算依旧从左向右执行，但是（a++）的值为a但是a会进行a += 1；具体过程如下：

例 ： b = a + (a++) + (++a);//5

先执行b = a = 1；

然后执行b += （a++） = 2；但是此时a = 2；这里就可以说明该语句中（a++）执行的结果为（a++）的值为a但是a会加一；

最后执行b += (++a);先执行++a，此时a = 3，所以b = 2 + 3 = 5；
